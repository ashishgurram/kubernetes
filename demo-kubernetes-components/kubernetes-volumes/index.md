# Kubernetes Volumes: Example Index

This folder contains example YAML manifests demonstrating different types of volume usage in Kubernetes. Follow the order below for a logical learning path:

1. `persistent-volumes.yaml` — Define persistent volumes (PVs) in the cluster.
2. `persistent-volume-claims.yaml` — Create persistent volume claims (PVCs) to bind to PVs.
3. `storage-class.yaml` — Define a storage class for dynamic provisioning.
4. `pods-with-volume.yaml` — Example of a pod using a volume.
5. `deployment-with-multiple-volumes.yaml` — Example of a deployment using multiple volumes.

---

Apply each manifest in order to see how Kubernetes volumes work from basic to advanced scenarios.
